BIP-157: Compact Filters (Watch-Only Wallet)

Explanation

A fully browser-based watch-only Bitcoin wallet built on BIP-157/BIP-158 compact block filters: block headers are validated locally (proof of work, difficulty retargets, median-time-past), every filter is verified against its commitment chain and matched against your watched addresses — all in WebAssembly, in this page.

Chain data is fetched from a block-dn server (configurable below; you can point it at your own). The server only ever learns which block ranges you download — never which addresses you are watching. Headers and filter headers are cached in the browser's Origin Private File System, so subsequent visits resume instead of re-syncing.

Sources:
