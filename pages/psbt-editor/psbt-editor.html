BIP-174: PSBT Editor

Explanation

Live editor for BIP-174 Partially Signed Bitcoin Transactions. Paste a base64-encoded PSBT in the top textarea, or build one from scratch. Every change is round-tripped through btcutil-js's WASM psbt.encode/decode on every keystroke, so the JSON view and base64 always reflect the editor below.

Sources:

  • BIP-174 specification
  • btcutil-js library
