Cryptography Toolkit

A web-based collection of cryptography tools for schemes/algorithms used in Bitcoin and LND.

This toolkit has been built with educational purposes in mind!
It is meant to play around with different schemes and algorithms to understand how they work.
However, you must be extremely careful when using real/live/mainnet data/keys/credentials!
A web browser usually is not a safe environment to either create strong cryptographic keys and/or paste sensitive information into. So consider yourself warned.

Tools

  • Elliptic Curve Cryptography / Key Pair
  • Bitcoin
    • Bitcoin Block Parser
    • Transaction Creator
    • Wallet Import helper
    • BIP-32: Hierarchical Deterministic Wallet
    • BIP-157: Compact Filters
    • BIP-174: PSBT Editor
    • BIP-322: Generic Signed Message Format
    • BIP-327: MuSig2 for BIP340-compatible Multi-Signatures
    • BIP-340: Schnorr Signatures
    • BIP-380: Output Script Descriptors
  • LND
    • aezeed Cipher Seed Scheme
    • Macaroons
  • Other
    • Shamir's Secret Sharing Scheme
    • Encoding/Decoding
